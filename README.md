# 機車駕照應考練習

一個純前端的台灣機車駕照筆試練習網站，可直接開啟 `index.html` 使用。

## 內容

- 一般筆試題庫：交通部公路局「機車筆試題庫_中文1150101」，共 804 題。
- 危險感知影片題：交通部公路局「機車危險感知影片選擇題-中文版1130816」，共 126 題。
- 題圖：自官方 PDF 擷取，共 148 張。
- 錯題本：答錯會自動加入瀏覽器本機錯題本，答對錯題後自動移除。

## 使用

直接用瀏覽器開啟 `index.html`。網站不需要登入、後端或資料庫。

## 更新題庫

官方 PDF 放在 `data/`，轉換腳本在 `scripts/import_questions.py`。下載新版 PDF 後，覆蓋對應檔案並重新執行轉換腳本，即會產生：

- `questions.json`
- `questions-data.js`
- `assets/questions/` 內的題圖
